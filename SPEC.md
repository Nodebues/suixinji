# 随心记 - 微信小程序规格说明书

## 1. 项目概述

- **项目名称**: 随心记
- **类型**: 微信小程序
- **核心功能**: 记录日常点滴，支持文字 + 图片
- **目标用户**: 个人日常记录

## 2. 功能清单

### 2.1 笔记列表
- 首页展示笔记列表（按时间倒序）
- 每条显示：时间、摘要、图片缩略图（如果有）
- 支持下拉刷新、上拉加载更多
- 空状态引导

### 2.2 新建笔记
- 文字输入框（支持多行）
- 添加图片按钮（最多9张）
- 图片支持拍照或从相册选择
- 发布按钮
- 返回时自动保存草稿

### 2.3 笔记详情
- 查看单条笔记完整内容
- 大图预览图片（可滑动）
- 删除笔记

### 2.4 数据存储
- 本地存储（wx.setStorage）
- 数据结构：
```json
{
  "id": "uuid",
  "content": "文字内容",
  "images": ["本地路径数组"],
  "createdAt": "时间戳",
  "updatedAt": "时间戳"
}
```

## 3. 页面结构

```
pages/
├── index/          # 笔记列表
│   ├── index.wxml
│   ├── index.wxss
│   ├── index.js
│   └── index.json
├── detail/         # 笔记详情
│   ├── detail.wxml
│   ├── detail.wxss
│   ├── detail.js
│   └── detail.json
└── editor/         # 编辑器
    ├── editor.wxml
    ├── editor.wxss
    ├── editor.js
    └── editor.json
```

## 4. UI 设计

- **风格**: 简约清新，仿日记本
- **配色**: 
  - 背景: #fafafa (浅灰白)
  - 卡片: #ffffff
  - 文字: #333333
  - 强调: #5B8FF9 (蓝色)
- **字体**: 系统默认
- **间距**: 16rpx 基础单位

## 5. 技术栈

- 微信小程序原生开发
- WXML + WXSS + JS
- 本地存储
- 图片选择器 (wx.chooseMedia)
