# 随心记

一款简洁的微信小程序，用于记录日常生活点滴。

## 功能特性

- 📝 **笔记记录** - 支持文字和图片记录
- 🌤️ **天气查询** - 实时获取当前位置天气信息
- ☀️ **日出日落** - 查看日出日落时间及日照时长
- 📊 **温度曲线** - 24小时温度变化趋势
- 💨 **左滑删除** - 便捷的笔记管理方式

## 预览

### 首页
展示笔记列表，支持左滑删除笔记，点击进入详情/编辑。

### 天气页
- 当前温度、天气状况
- 湿度、风速、体感温度
- 24小时温度曲线（可点击查看详情）
- 日出日落时间、日照时长

### 笔记详情
- 点击内容即可编辑
- 支持图片预览

## 技术栈

- 微信小程序原生开发
- Open-Meteo API（天气数据）
- Nominatim API（地理编码）

## 使用说明

1. 克隆仓库
```bash
git clone https://github.com/Nodebues/suixinji.git
```

2. 使用微信开发者工具打开项目

3. 配置位置权限
   - 在微信公众平台申请开通位置接口权限

## 项目结构

```
suixinji/
├── app.js              # 小程序入口
├── app.json            # 小程序配置
├── app.wxss            # 全局样式
├── pages/
│   ├── index/          # 首页（笔记列表）
│   ├── detail/         # 笔记详情
│   ├── editor/         # 笔记编辑
│   └── weather/        # 天气页面
└── project.config.json # 项目配置
```
